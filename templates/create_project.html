Project Description - The Key to Success

This is what matters most! Your description is the only information our agent uses to find relevant papers.

How it works: The agent extracts keywords from your description to search for papers. Poor descriptions = poor recommendations.

Best practices:

  • Be specific - include relevant keywords and technical terms
  • Write naturally - describe your work or research interests in 3-4 sentences
  • Provide enough detail - too brief won't give enough context
  • Experiment - try different phrasings to steer recommendations

Current limitation: The system only works with keyword extraction. It can't handle requests like "recent papers" or "highly cited papers."

Bottom line: More precise, detailed descriptions = better recommendations. This is your only way to guide the agent, so make it count!
